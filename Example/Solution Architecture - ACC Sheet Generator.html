Solution Architecture

A clear separation of concerns between client-side interactions and server-side processing.

Front End
Browser UI (HTML/CSS) Responsive dashboard with 3-step workflow and progress tracking.
Client Logic (app.js) Manages user interaction, tree navigation, and triggers server commands.
APS OAuth Access Token Held in a secure browser session for API authorization.
Back End
Easy Auth Proxy (Azure) Enforces Microsoft Entra ID Sign-on before request hits the code.
Express.js Server (server.js) Handles secure API routing, file extraction, and ACC file transfers.
ZIP Processing Engine Memory-resident extraction of Revit PDF exports for individual access.
